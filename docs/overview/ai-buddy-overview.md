# AI Buddy

## What It Does

AI Buddy creates optimized AI prompts customized to your personality type and the specific task you're trying to accomplish. By understanding your personality traits (based on 16personalities.com types) and applying established prompt frameworks, tailored to the task at hand, it generates high-quality prompts that will produce better results from AI systems.

## How It Works

This agent follows a personalized workflow:

1. **Collects your personality information** - Select your 16personalities.com personality type and optionally enter your trait percentages
2. **Understands your task** - Describe what you're trying to accomplish
3. **Selects appropriate frameworks** - Chooses the best prompt framework (SMART, RICE, COAST, etc.) for your task
4. **Adopts a complementary role** - Either amplifies your personality traits or balances them with complementary traits
5. **Asks clarifying questions** - Tailored to your personality type and using the selected framework
6. **Suggests output formats** - Recommends relevant formats for your specific task
7. **Creates an optimized prompt** - Generates a prompt engineered for better results
8. **Delivers formatted results** - Presents the final prompt in your preferred format

## Required Setup

Before using this agent, ensure you have:

1. Created a **Personality Types data source** containing information about the 16 personality types
2. Uploaded the **prompt_frameworks_guide.md** document with information about prompt frameworks

## How to Use

1. Run the agent
2. Select your personality type from the dropdown menu
3. Optionally enter your trait percentages (e.g., "I - 65%, N - 64%, T - 84%, P - 57%, A - 84%")
4. Describe your task in detail
5. Engage with the clarifying questions to refine understanding
6. Select your preferred output format
7. Receive your personalized, optimized prompt
